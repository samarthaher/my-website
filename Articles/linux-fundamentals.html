🐧 Linux Cheat Sheet

📁 File Navigation

  • ls -la - List all files with permissions and hidden files
  • cd /path - Change directory
  • cd .. - Move one level up
  • pwd - Print working directory
  • tree - Show directory tree (may require install)

📝 File Management

  • touch file.txt - Create an empty file
  • cp a.txt b.txt - Copy file
  • mv a.txt b.txt - Move/rename file
  • rm file.txt - Delete a file
  • nano file.txt - Edit file using nano

📦 Directory Management

  • mkdir newdir - Create a new directory
  • rmdir dir - Remove empty directory
  • rm -rf dir - Delete directory and contents
  • du -sh * - Disk usage of folders

📜 File Viewing

  • cat file - View file content
  • less file - Scroll through file
  • tail -f log.txt - Watch file live
  • head -n 10 file - First 10 lines
  • wc -l file - Count lines

🔒 Permissions

  • chmod +x script.sh - Make file executable
  • chmod 755 file - Set permissions
  • chown user file - Change file owner
  • ls -l - View file permissions

🔎 Searching

  • grep 'text' file - Search in file
  • grep -r 'text' dir - Recursive grep
  • find / -name "file.txt" - Find file by name
  • locate file.txt - Find file using database

⚙️ Processes

  • ps aux - Show all processes
  • top - Task manager
  • kill PID - Kill process by ID
  • killall name - Kill by name
  • bg / fg - Resume jobs in background/foreground

🌐 Networking

  • ip a - Show network interfaces
  • ping 8.8.8.8 - Test connectivity
  • curl ifconfig.me - Show public IP
  • netstat -tuln - Listening ports
  • ss -tuln - Alternative to netstat

📦 Package Management

  • sudo apt update - Update package list (Debian)
  • sudo apt install curl - Install package
  • sudo apt remove nginx - Remove package
  • yum install git - Install via yum (RHEL/CentOS)

🖥️ System Info

  • uname -a - System information
  • uptime - How long system is running
  • df -h - Disk space usage
  • free -m - Memory usage
  • lscpu - CPU info

👤 User Management

  • whoami - Show current user
  • adduser newuser - Create new user
  • passwd user - Change user password
  • usermod -aG sudo user - Add user to sudo group
